## 🔭 Qu’est-ce que Polaris ?

Polaris est un **outil d’exploration personnelle** basé sur une simulation astronomique (positions képlériennes réelles) et une lecture symbolique des cycles historiques (psychohistoire). Il ne donne pas de vérité absolue, mais propose une **lecture expérimentale** pour aider chacun à se situer dans le temps, l’espace et les grands mouvements collectifs.

### Caractéristiques

- 🌌 **Carte céleste personnalisée** (signe solaire, lunaire, ascendant)
- 🌍 **Visualisation 3D** du système solaire (orbites, planètes, étoiles)
- 📜 **Terminal Janus** – commandes `profil`, `theme` et questions à l’oracle
- 🔄 **Moteur psychohistorique** (cycles PHE, Kondratiev, Turchin)
- ⚡ **Mode expérimental** – aucune prédiction absolue, usage symbolique

## 🧭 Comment utiliser

1. **Onboarding** : entrez votre date de naissance, heure et géolocalisation (ou cliquez “Explorer sans profil”).
2. **Carte céleste** : visualisez vos signes dans le cercle zodiacal.
3. **Terminal** :
   - Tapez `profil` pour voir vos données
   - Tapez `theme` pour rappeler votre carte céleste
   - Posez n’importe quelle question → réponse contextuelle
4. **Exploration libre** : faites glisser la vue 3D, utilisez le curseur temporel pour simuler le passé/futur.

## 🛠️ Technologies

- **Three.js** – rendu 3D du système solaire et de la sphère céleste
- **Mécanique céleste** – positions planétaires basées sur les équations de Kepler (éphémérides VSOP87 simplifiées)
- **JavaScript moderne** – ES modules, CSS Grid, canvas 2D
- **Stockage local** – localStorage pour conserver votre profil
